# Contributing

Thanks for your interest in improving the **Generative AI Seminar Projects** repo!

## How to Contribute

1. **Fork & Clone**

   ```bash
   git clone https://github.com/Developer-Linus/generative_ai_ouk.git
   cd generative_ai_ouk
   ```
2. **Create a Branch**

   ```bash
   git checkout -b feature/my-change
   ```
3. **Make Changes**

   ```bash
   git add .

   ```
